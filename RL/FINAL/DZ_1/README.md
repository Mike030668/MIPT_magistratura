# Контекстные бандиты

## Описание датасета и применение контекстных бандитов

Датасет BibTeX Dataset предствлен на [сайте ](http://manikvarma.org/downloads/XC/XMLRepository.html)

- данные в датасете рассматривают различные задачи по категоризации веб-страниц, рекомендации связанных веб-страниц и рекомендации товаров. Они включают в себя мультимодальные наборы данных и наборы данных, в которых метки имеют текстовые характеристики.

- Многомерные признаки (контекст):
  - Каждый документ представлен вектором признаков, что позволяет использовать эти признаки как контекст для контекстных бандитов.

  - Признаки могут включать такие данные, как слова или темы, которые описывают документ, что является типичным контекстом для контекстных бандитов.

- Многомерные метки (действия):
  - Каждая метка представляет собой тег, связанный с документом.
  - В задаче контекстных бандитов действиями могут быть различные теги, которые агент должен выбрать на основе контекста (признаков документа).


- Задачи и цели при использовании BibTeX Dataset
  - Задача: На основе признаков документа (контекста) выбрать наилучшие теги (действия) для этого документа, чтобы максимизировать общее вознаграждение (например, соответствие тегов).
  - Цель: Построить и протестировать алгоритмы контекстных бандитов, чтобы определить, какой алгоритм лучше всего подходит для максимизации вознаграждения в данной задаче.
